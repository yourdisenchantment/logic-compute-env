# Распределенное программирование

## Распределенное программирование с использованием MPI

С помощью библиотеки MPI посчитайте количество чисел в диапазоне `[l, r]` которые делятся на 11, 13 или 17.

Процесс с номером 0 должен прочитать границы и разослать их всем остальным процессам, посчитать интервал, который
достался ему и просуммировать количество чисел, которые нашли все процессы.

Остальные процессы получают интервал, считают количество чисел и отправляют в ответ найденную сумму
